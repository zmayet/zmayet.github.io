Bootcamp 2020: Car Animation

The duration of Bootcamp lasted 2 weeks which enabled us to develop a set of skills to develop our own unique project.In order to showcase our skills I had decided to develop a small animation of cars racing, which involved a variety of codes including: arrays, loops, if statements, randomness e.t.c. Once the user clicks anywhere on the screen this will spark a set of codes to be carried out, meaning the cars will not move unless the screen has been clicked. Therefore, when the screen has been clicked, random variables will be set for each car so every time the program is excecutes a different car will win the race. I had also added an animation background to give the effect of rain, as well as a message to appear when either car wins the race.
