This endeavour was insightful as it was filled with numerous learning opportunities and challenges. Beyond the enhancement of my pre-existing skills the project further developed my technical skills through learning and applying complex ideas within the website. For an in-depth account of the project can be found on Github.

RaceStar

The first coding endeavour carried out studying Computer Science at University was a project completed in “Bootcamp”. The project’s brief was open to creativity thus providing a fun and educational venture. I had decided to create an animation giving the two cars ‘random’ variables thus producing a new winner every time the "start" button was pressed.

Is the Film Industry Dying?

This module was all about displaying data visually. My main research question for this project was to explore a dataset to answer the question: “Is the movie industry dying?”. As I’d like to consider myself a movie fan this project was extremely enjoyable as I not only learnt new skills in coding a dataset to output visually, but ultimately learnt new facts and statistics of an industry I am quite fond of.

2D Game

Using Java I had created a 2D, platform style game. This project strengthened my knowledge and understanding within the programming language as I was able to revisit the foundations of the language as well as learn new skills such as incrementing a coin count, decrementing lives, using keys to control/ move the character.

Japan Vlog

During my 2023 trip to Japan I decided to explore video editing. I decided to record fragments of my holiday and compiled this videos/pictures using different video effects and visuals. This was an exciting new endevour as it opened doors to a creative side which I have not yet explored.
